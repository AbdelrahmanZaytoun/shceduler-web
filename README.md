# Interview Scheduler

## Requirements

- Backend server running you can find [here](https://github.com/AbdelrahmanZaytoun/scheduler-api/)
- NodeJS 16v (LTS)

## Setup

Install dependencies with `npm install`.

## Running Webpack Development Server

```sh
npm start
```

## Running Storybook Visual Testbed

```sh
npm run storybook
```
